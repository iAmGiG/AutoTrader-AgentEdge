## Recommendations

### Primary Recommendation: Hybrid Integration

Use weekly KAMA as **trend filter** for daily VoterAgent:

```text
IF weekly_kama_bullish:
    Execute daily MACD+RSI signals
ELSE IF weekly_kama_bearish:
    Reduce position size OR exit
ELSE (neutral):
    Hold current position
```

### Implementation Priority

1. **High Value**: Use for QQQ, IWM - KAMA outperforms
2. **Low Value**: Skip for SPY - MACD baseline is better
3. **Caution**: 3x ETFs (TQQQ, SOXL) have high drawdowns regardless

### Tax Efficiency Benefit

With 0.8 trades/year average, most gains qualify for **long-term capital gains** (held >1 year).

## Technical Implementation

### KAMA Formula

```python
efficiency_ratio = abs(close - close[lookback]) / sum(abs(close.diff()), lookback)
smoothing = (efficiency_ratio * (fast_sc - slow_sc) + slow_sc) ** 2
kama = kama_prev + smoothing * (close - kama_prev)
```

### Signal Logic

- **Bullish**: KAMA slope > 0 AND Fold MA slope > 0 AND price > both
- **Bearish**: KAMA slope < 0 AND Fold MA slope < 0 AND price < both
- **Neutral**: Mixed signals
