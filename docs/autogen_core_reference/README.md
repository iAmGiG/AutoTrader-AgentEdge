# AutoGen Core Reference Documentation

This directory contains reference documentation for key AutoGen Core classes and patterns used in the RH2MAS project. These documents serve as a quick reference for developers working with the codebase, eliminating the need to set up the conda environment just to check class interfaces.

## Contents

- [FunctionTool Reference](function_tool.md) - Documentation for the `FunctionTool` class used to create callable tools
- [Messaging Reference](messaging.md) - Documentation for message types like `SystemMessage`, `UserMessage`, etc.
- [OpenAI Client Reference](openai_client.md) - Documentation for the `OpenAIChatCompletionClient` class

## Purpose

These reference documents aim to:

1. Provide clear documentation on class interfaces and usage patterns
2. Serve as a quick reference for common AutoGen Core operations
3. Capture best practices observed in the RH2MAS codebase
4. Reduce dependency on environment setup for checking class interfaces

This documentation is based on AutoGen Core version 0.5.1, which is the version used in the RH2MAS project.
