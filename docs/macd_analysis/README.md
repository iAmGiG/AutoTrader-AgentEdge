# MACD Analysis Scripts

This directory contains scripts for analyzing MACD (Moving Average Convergence Divergence) behavior and data quality.

## Scripts

- `diagnose_macd_periods.py` - Diagnose MACD crossing periods and signal quality
- `find_macd_recovery_periods.py` - Find periods where MACD recovers from negative to positive
- `compare_macd_values.py` - Compare MACD line vs histogram values to understand differences

## Usage

These scripts are primarily used for:

1. Understanding MACD behavior in different market conditions
2. Diagnosing precision issues with MACD calculations
3. Optimizing MACD-based trading strategies

Example:

```bash
python scripts/macd/compare_macd_values.py
```
