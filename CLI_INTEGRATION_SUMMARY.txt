================================================================================
HOW CLI CURRENTLY WORKS (Entry Point Flow)
================================================================================

main.py (python main.py)
  ├─→ trade_assist()
  │    ├─→ OrchestratorFactory.create()
  │    │    └─→ Creates TradingOrchestrator with RealVoterStrategy
  │    │
  │    └─→ CLISession(orchestrator)
  │         └─→ Interactive REPL loop
  │
  └─→ User enters: "is AAPL a good buy?"
       │
       ├─→ CLISession.process_input()
       │    └─→ TradingOrchestrator.process_request()
       │         └─→ RealVoterStrategy.analyze()
       │              └─→ VoterAgent.evaluate_voting()  ⬅️ LINE 118
       │                   ├─→ Calculates MACD
       │                   ├─→ Calculates RSI
       │                   └─→ Returns consensus vote (BUY/SELL/HOLD)
       │
       └─→ CLI shows suggestion with position size & entry/stop/target

THE ISSUE:
  - RealVoterStrategy line 118 calls: voter.evaluate_voting()
  - But new method exists: voter.evaluate_ranked_voting()
  - New method uses: IndicatorRegistry + RankedVoterManager
  - Old method ignores: New ranked voter system + presets
